# Security Policy

`dreams` is a static public research showcase, but security reports are still welcome.

## Please report

- site-level vulnerabilities in the static surface under `site/`
- accidental disclosure of private paths, credentials, or unpublished material
- bundle-generation issues that could publish unintended files

## Please do not use public issues for sensitive reports

Send responsible-disclosure reports to <mailto:jackgaff@umich.edu> with:

- a short description of the issue
- reproduction steps or screenshots when relevant
- the affected file or page
- any suggested mitigation

I’ll acknowledge receipt and work to confirm the issue quickly.
