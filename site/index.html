The Validity Mirage

When AI agents compress long conversations to save memory, they can lose track of which task they're solving while still producing confident answers. We call this a validity mirage.

Abstract

Context compression can look valid while the governing task has drifted. This research program studies the structural failure, proves where each layer of reasoning breaks under memory pressure, and provides a tool (tropical-mcp) to detect and prevent it in real AI coding agents. Every claim links to committed artifacts and deterministic replay data.

The Problem

AI agents silently lose their governing task under memory pressure, then keep working as if nothing happened.

When you work with an AI coding agent on a complex task, you give constraints across many messages: "use async I/O", "target Python 3.10", "don't break the public API". As the conversation grows, the client silently compresses older messages to stay within its context window.

The agent keeps working confidently. But it has lost 3 of your 7 requirements. You catch the drift, re-explain, and the agent refactors. This is the validity mirage: the context looks fine, the agent feels fine, but critical information is gone.

The Mirage Shelf

Under memory constraints, correctness and justification come apart. There is a measurable zone where answers survive but the reasons behind them don't. A system can be right without knowing why.

Mathematical Foundations

Core Contract

d_pre >= k

Guarded compaction is certified safe only when the pivot retains its required predecessor depth.

Frontier Feasibility

W[k] = -infinity → infeasible

If the k-slot frontier is negative infinity, no valid completion exists for that retained context.

Raw Validity

raw = max(primary_full, decoy_full)

Answerability alone can stay high even when pivot identity has silently changed.

Mirage Gap

delta = raw - pivot_preservation

Large positive gap indicates a validity mirage regime rather than true semantic stability.

Run It Yourself

tropical-mcp gives AI coding agents explicit tools to detect and prevent the validity mirage. Register it as an MCP server, then inspect every compression step.

Prerequisites

Requires Codex or Claude Desktop with MCP support, plus uv (Python package manager).

  uv --directory ~/tropical-mcp run tropical-mcp
