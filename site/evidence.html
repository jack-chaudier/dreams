Reproduce

Verification paths

Install and register tropical-mcp, then choose between the minimal smoke path and the fuller reviewer workflow. The public certificate on this page is a comparison target for your local run, not a substitute for the reviewer workflow itself.

  • Smoke test: confirm the runtime with runtime_info(), then run compact_auto(...) and certificate(...).
  • Research workflow: run diagnose(...) to inspect feasible slots, capture a context_anchor(...), then compare compact_auto(...), certificate(...), and telemetry_summary(...).
  • Compare the resulting certificate and replay behavior against the public artifacts on this page.

# minimal smoke path
runtime_info()               # check your setup
compact_auto(                # compress with policy selection
  messages=messages,
  token_budget=45,
  k_target=2,
  mode="adaptive",
)
certificate(                 # generate portable audit record
  messages=messages,
  token_budget=45,
  k=2,
)

# fuller reviewer workflow
runtime_info()               # check your setup
diagnose(messages=messages, k_max=2)    # find compression issues
context_anchor(messages=messages, k=2)  # mark protected messages
compact_auto(                # compress with policy selection
  messages=messages,
  token_budget=45,
  k_target=2,
  mode="adaptive",
)
certificate(                 # generate portable audit record
  messages=messages,
  token_budget=45,
  k=2,
)
telemetry_summary(limit=5)   # review retention behavior
