Reproduce

Verify locally

Install and register tropical-mcp, then choose between the minimal smoke path and the fuller reviewer workflow. The public certificate is a comparison target — not a substitute for running it yourself.

Smoke test — 3 steps
runtime_info()               # check your setup
compact_auto(
  messages=messages,
  token_budget=45,
  k_target=2,
  mode="adaptive",
)
certificate(                 # portable audit record
  messages=messages,
  token_budget=45,
  k=2,
)
Full reviewer workflow — 5 steps
runtime_info()               # check your setup
diagnose(messages=messages, k_max=2)
context_anchor(messages=messages, k=2)
compact_auto(
  messages=messages,
  token_budget=45,
  k_target=2,
  mode="adaptive",
)
certificate(messages=messages, token_budget=45, k=2)
telemetry_summary(limit=5)   # review retention behavior
